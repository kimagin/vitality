## Installation

Clone the repository via _Terminal_ :

```bash
git clone https://github.com/kimagin/TailwindCSS-boilerplate .
```

 After cloning the repository in your folder, install the dependencies via npm :

```npm
npm install
```

## Developement

```npm
npm run dev
```

## Build

```npm
npm run build
```
